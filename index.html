Front-End and Back-End Development Terms

JavaScript
JavaScript is a programming language used in front-end development add interactivity and dynamic behaviour to web pages and applications.
User Interface(UI)
The user interface in front-end development refers to the visual elements and design of a website or application that users interact with.
Cascading Style Sheets(CSS)
CSS is a style sheet language used in front-end development to control the layout and prsentation of web pages, including element's colors, fonts, and spacing.
Python
It is a popular programming language that was created by Guido van Rossum, and it is used for web development, software development, mathematics, etc. It was released in 1991
